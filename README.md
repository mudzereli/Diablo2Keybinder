## D2 Keybinder

A tool to help make keybinds in D2 more modern.


| Old Key                                    | New Key |
|--------------------------------------------|---------|
| F1 (Skill 1)                               | Q       |
| F2 (Skill 2)                               | W       |
| F3 (Skill 3)                               | E       |
| F4 (Skill 4)                               | R       |
| F5 (Skill 5)                               | T       |
| F6 (Skill 6)                               | F       |
| F7 (Skill 7)                               | G       |
|--------------------------------------------|---------|
| F8 (Skill 8 - Recommended Town Portal Key) | A       |
|--------------------------------------------|---------|
| Q (Quests)                                 | Z       |
| A (Attributes)                             | X       |
| T (Skills)                                 | C       |
| I (Inventory)                              | V       |
|--------------------------------------------|---------|
| W (Switch Weapons)                         | S       |
| R (Run/Walk)                               | D       |
| Alt (Alt = Temp Show, B = Toggle Showing)  | B       |
